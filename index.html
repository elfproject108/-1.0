理性人文主義・AI 對話提示詞

這個小頁面會自動從 GitHub 讀取你的 README 中那段「完整提示詞」，讓你或朋友只要按兩個按鈕，就能快速複製並貼到 ChatGPT、Claude、DeepSeek 或 Copilot 的第一個訊息中使用。

使用步驟

  1. 按下面的「載入 GitHub 上的提示詞」，稍等數秒。
  2. 檢查下方文字框內容是否正確。
  3. 按「複製整段 Prompt」。
  4. 開啟 ChatGPT／Claude／DeepSeek／Copilot，新建對話，把這段完整貼上並送出。
  5. 看到模型回覆「明白」或類似確認後，再開始問你真正的問題。
